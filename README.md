# Dom's Portfolio

Personal portfolio website of **June Dominic G. Laurente** — a Filipino Full-Stack Web Developer and IT student, built with a bold **neo-brutalism** design aesthetic.

## Tech Stack

- **Frontend:** Vue 3 (Composition API, `<script setup>`)
- **Build Tool:** Vite 8
- **Backend:** Firebase (Firestore, Auth, Analytics, Hosting)
- **Vue-Firebase Integration:** VueFire 3
- **Styling:** Custom CSS — Neo-Brutalism theme
- **Animation:** AOS (Animate On Scroll)
- **Icons:** Font Awesome 6
- **Fonts:** Lexend + Space Grotesk

## Sections

| Section | Description |
|---|---|
| **Hero** | Profile, visitor counter, call-to-action, tech stack marquee |
| **About** | Bio, identity card, skills, academic timeline |
| **Projects Vault** | Filterable project grid with detail modals |
| **Certificates** | Credentials from CSU, Simplilearn, Udemy, Codec |
| **Contact** | Social links (Facebook, GitHub, LinkedIn, Email) |
| **Arcade** | Pixel farm game built with Kaplay |

## Getting Started

```bash
npm install
npm run dev      # development server
npm run build    # production build
npm run preview  # preview production build
```

## Deploy

```bash
npm run deploy   # Firebase Hosting
```
